Title: CIT Laurentides Bus (MonTransit)
----------
Short description: CIT Laurentides bus information for MonTransit BETA.
----------
Full description:
BETA: TEST IN PROGRESS
To leave the test: http://goo.gl/KoTkAw

This application adds CIT Laurentides bus information to MonTransit BETA.

Once this application is installed, the MonTransit BETA app will display bus information (schedule...).

This application only has a temporary icon and no screen: download the MonTransit BETA app (free) in the "More ..." section bellow or by following this Google Play link http://goo.gl/pCk5m

You can install this application on the SD card but it is not recommended.

The information comes from the GTFS file provided by the Agence Métropolitaine de Transport.
http://www.amt.qc.ca/developers/

This application is free and open-source:
https://github.com/mtransitapps/ca-laurentides-citla-bus-android

This app is not related with the Agence Métropolitaine de Transport and CIT Laurentides.
